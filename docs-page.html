Introduction Last updated: 29th June 2022

Hey there! Welcome to the LlamaBot docs and info site! LlamaBot is a reactive discord bot that does llama related stuff (plus a lot more!) No donator only, or voter only commands. Everyone gets the same access to LB!

Installation

LlamaBot is super easy to install into your server. Give it a couple clicks and you'll have it installed!

Note

This guide assumes you have no knowledge of adding discord bots to your server. Setup is same as usual, so skip this bit if you already know how it all works

Adding and setting permissions

Adding is quite simple, and easy to do. 1. Click the Add to Discord button at the top right of this page 2. Sign into discord 3. Set all the permissions you wish to keep/remove

Warning

We reccomend that you keep the permissions as the pre-set suggests, as turning some off might stop the bot from working properly.

More specific bot permissions

Once LlamaBot is added, Discord provides a much richer setup for LlamaBot permissions To access these settings, head to: <Server Settings > Integrations > LlamaBot>. From here, you're able to customize who can use which command, and in what channel!

Privacy FAQ's

What information is stored about me?

The LlamaBot database is quite small in size. We track the following:
- Your user name (Used in display, when you run commands)
- How many messages you've sent (which we use to add coins to your balance)
- Your user id (Used to link your user name and balance)

We don't store:
- Passwords
- Server (Guild) names/info
- Other information about you

Who can view these details?

Your user name and balance can be publicly viewed on the leaderboard by anyone.
Other than this, no one can view your information but the bot developer and you.
